use the arrow keys to move the cursor

:> press ':' (shift + ';') to use one of the editor commands :

     ->	w    : to write your changes into the file

     ->	q    : to quit the program

     ->	goto options :

         > type the number of the line and it will go to that line

         > type "start" to go to the start of the file

         > type "end" to go to the end of the file

     ->	man  : to open the manual

:> press '$' (shift + '4') to execute terminal commands (cmd only)

you can use multiple commands at once :
-> example :typing ":wq" will write the changes into the file then quit the program

press 'q' to exit the manual page
